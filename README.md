# Web Development Programs

## Description
- This repository contains basic HTML programs.
- These programs are created for learning purpose and academic submission.

## Files

- CO-1
  - Contains programs related to Course Outcome 1.

- CO-2
  - Contains programs related to Course Outcome 2.

- ERP.html
  - HTML program to display ERP timetable using table tags.

- Food and Hobbies.html
  - HTML program showing favourite foods using ordered list.
  - HTML program showing hobbies using unordered list.

- README.md
  - Contains information about the repository and files.

## Technology Used
- HTML

## Usage
- Open the HTML files in any web browser to view the output.

## Objective
- To understand basic HTML structure.
- To learn lists and tables in HTML.
